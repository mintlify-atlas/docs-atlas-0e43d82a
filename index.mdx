## Key Features

Prism Launcher is a powerful, open-source Minecraft launcher that makes managing your Minecraft experience simple and efficient.

<CardGroup cols={2}>
  <Card title="Multiple Instances" icon="layer-group" href="/guides/instances">
    Create and manage unlimited Minecraft installations with different versions, mods, and configurations
  </Card>
  <Card title="Mod Platform Integration" icon="puzzle-piece" href="/mod-platforms/modrinth">
    Browse and install mods directly from Modrinth, CurseForge, ATLauncher, FTB, and Technic
  </Card>
  <Card title="Java Management" icon="coffee" href="/guides/java-setup">
    Automatically download and manage Java versions for optimal Minecraft performance
  </Card>
  <Card title="Cross-Platform" icon="desktop" href="/installation">
    Available for Windows, Linux, and macOS with native support for each platform
  </Card>
</CardGroup>
